# Dotfiles

A collection of my dotfiles so I can easily access them from anywhere.

# Prerequisites

## `.cargo/config.toml`

Configures: Cargo

Requires:
- `sccache` (via Cargo)

## `.config/helix/`

Configures: Helix editor

## `.config/neofetch`

Configures: `neofetch`

## `.config/nushell`

Configures: Nushell

Requires:
- Starship prompt (via cargo)

## `.starship.toml`

Configures: Starship prompt
